Teste de Funcionalidade de Erros de Validação

Teste do Hook useFieldErrors

Este teste simula diferentes cenários de erro que a API pode retornar:

Cenários de Teste

1. Erros de Campo Específicos

Simula quando a API retorna erros para campos específicos (ex: email já existe, senha muito fraca)

2. Erro Geral

Simula quando a API retorna um erro geral (ex: servidor indisponível)

3. Erros Mistos

Simula quando a API retorna tanto erros de campo quanto erro geral
